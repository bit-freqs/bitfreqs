
# bitFreqs

game for global game jam 17

# [https://bit-freqs.github.io/bitfreqs/](Play the game!!!!!!)

## how

### install

```
git clone https://github.com/bit-freqs/bitfreqs.git
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
